# BackSeat Chronicles
Real-Time Storytelling & Social Engagement Platform

## About
A scalable storytelling platform supporting multimedia publishing,
personalized feeds, and real-time social engagement built with 
Java Spring Boot microservices architecture.

## Tech Stack
- Backend: Java 17, Spring Boot 3.x, Spring Security, Spring Data JPA
- Database: MySQL, Redis (caching)
- Messaging: Apache Kafka
- Cloud: AWS S3
- Frontend: Angular 17
- DevOps: Docker, Kubernetes, Jenkins
- Tools: Swagger, Postman, Git

## Key Features
- 10+ RESTful APIs with sub-200ms response times
- JWT + RBAC authentication and authorization
- Real-time engagement: likes, comments, bookmarks, follow/unfollow
- Multimedia uploads (image, video, voice) via AWS S3 — 500+ daily uploads
- Redis caching reducing API response time by 35%
- Apache Kafka event-driven notification system decoupling 5+ services

## API Documentation
Built with Swagger/OpenAPI — covers all endpoints with request/response examples
